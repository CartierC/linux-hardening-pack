# Linux Hardening Pack

Standardized Linux hardening scripts and security configurations designed to demonstrate practical Linux administration, security baseline enforcement, verification workflows, and rollback awareness.

## Purpose

This repository showcases a practical Linux hardening project for systems administration, cloud, IT support, and DevOps portfolio use. It focuses on turning common Linux security best practices into repeatable scripts, baseline configurations, and verification steps.

## What This Repo Demonstrates

- SSH hardening
- UFW firewall verification
- Fail2ban service verification
- Kernel and sysctl hardening checks
- Security-minded shell scripting
- Rollback-aware operational thinking
- CI-based script linting and validation
- Organized repo structure for professional portfolio presentation
